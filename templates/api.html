API

Request path: /api/create
Method: GET or POST
Input parameters:
url - The desitnation url
minutes - The time in minutes before expiration (min: 0, max: 1440000)
visits - The number of visists before expiration
Results are returned as a JSON encoded string. The result object will contain either:
id - The id to append to http://vapurl.com/
or
error - An error message

Example
Request: http://vapurl.com/api/create?url=google.com&minutes=1&visits=1
Response: {"id":"6r783omub4"}
